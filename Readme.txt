Это лабораторная работа № 3  ученика Московского инженерно физического института Гаины Георгия.

В данной лабораторной работе реализованы:
    структура данных под названием словарь(хэш таблица).
    итераторы к словарю и связному списку.
    Last Rcently Used Cache.
    Unit тесты к созданным типам данных.
    Нагрузочные тесты, позволяющие сравнить время выполнения аналогичных операций на
        мною созданной структуре данных и std::unordered_map.
    
Лабораторная работа оснащена графическим пользовательским интерфейсом для словаря и кэша,
    они размещены в разных файлах.
    Запускать UI нужно из соответствующих каталогов build.
Перед запуском main необходимо собрать все подпроекты в соответствующем порядке
    (файловая система образует дерево, соответствующий порядок снизу-вверх)
